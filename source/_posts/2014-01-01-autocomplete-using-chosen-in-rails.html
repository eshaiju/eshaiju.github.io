In one of my recent project, i need to implement a autocomplte in active_admin interface to select one or more users from user list. I tried googling on this and zeroed in on the library Chosen.Chosen by default supports multiple select, selected state.Chosen is available as a gem for Ruby on Rails which fits well with Asset Pipeline.
Include Chosen gem in your Gemfile. {%codeblock%} gem ‘chosen-rails’ {%endcodeblock%} Once the gem is installed, include chosen javascript assets to your js file. {%codeblock%} //= require chosen-jquery {%endcodeblock%} In /app/admin/modelname.rb {%codeblock%} ActiveAdmin.register Modlename do #Customize create and edit form form do |f| f.inputs do f.input :name f.input :othermodel, :input_html => { :class => "chosen-input" } # other model with has_many relation ship end f.buttons end end {%endcodeblock%} in active_admin.js {%codeblock%} $(document).ready(function(){ $(".chosen-input").chosen(); }); {%endcodeblock%} if you are using coffee script instead of js then use this {%codeblock%} $ -> # enable chosen js $('.chosen-input').chosen allow_single_deselect: true no_results_text: 'No results matched' {%endcodeblock%} Then, include Chosen stylesheet assets to your css file. {%codeblock%} *= require chosen {%endcodeblock%} To include more details check Here
Thats it, you will have a nice autocomplte with multiselect . Happy coding…
